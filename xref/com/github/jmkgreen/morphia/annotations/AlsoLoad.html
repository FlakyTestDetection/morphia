8   /**
9    * <p>Annotation which helps migrate schemas by loading one of several possible properties
10   * in the document into fields or methods.  This is typically used when a field is renamed,
11   * allowing the field to be populated by both its current name and any prior names.</p>
12   * <p/>
13   * <ul>
14   * <li>When placed on a field, the additional names (document field) will be checked
15   * when this field is loaded.  If the document contains data for more than one of the names,
16   * an exception will be thrown.
17   * <li>When placed on a parameter to a method that takes a single parameter, the method
18   * will be called with the data value.  As with fields, any ambiguity in the data (multiple
19   * properties that would cause the method to be called) will produce an exception.  However,
20   * {@code @AlsoLoad} on a method parameter *can* be used to override field names and "steal" the
21   * value that would otherwise have been set on a field.  This can be useful when changing the
22   * type of a field.</li>
23   * </ul>
24   * <p/>
25   * (orig @author Jeff Schnitzer <jeff@infohazard.org> for Objectify)
26   *
27   * @author Scott Hernandez
28   */
29  @Retention(RetentionPolicy.RUNTIME)
30  @Target({ElementType.FIELD, ElementType.PARAMETER})
31  public @interface AlsoLoad {
32      String[] value();
33  }
